# CameraDemo
实现对相机的自定义，并且可以在全屏显示时，正确截取图片中间区域。

## Features
 * 实现全屏显示，兼容几乎所有手机；
 * 实现正确截取中间区域
 * 实现自动对焦，打开闪光灯，调焦距等
